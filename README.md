# Club Management System

A full stack web application developed using React and Django.

## Features

* Member management
* Event management
* Meeting scheduling
* User-friendly dashboard

## Technologies Used

### Frontend

* React
* Axios
* CSS

### Backend

* Django
* Django REST Framework

## How To Run Project

### Frontend

```bash
npm install
npm start
```

### Backend

```bash
pip install -r requirements.txt
python manage.py runserver
```
